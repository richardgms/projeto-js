Regras do Jogo

  1. O jogo utiliza seis cores diferentes: vermelho, amarelo, azul, rosa, preto e branco
  2. Um jogador cria uma sequência secreta de cores que o outro jogador deve adivinhar
  3. Os jogadores têm até 10 tentativas para descobrir a sequência correta
  4. Após cada tentativa, são fornecidas dicas através de:
    • Bola preta: indica uma cor correta na posição correta
    • Bola branca: indica uma cor correta na posição errada
  5. Use as dicas das tentativas anteriores para fazer deduções lógicas sobre:
    • Quais cores estão na sequência
    • Onde cada cor deve ser colocada
  6. O jogo continua até que:
    • O jogador adivinhe corretamente a sequência
    • Todas as 10 tentativas sejam usadas
